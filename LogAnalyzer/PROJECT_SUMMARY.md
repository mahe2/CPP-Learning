## 📚 掌握的 C++ 技术

### 1. 现代 C++ 特性 (C++17)
- **智能指针**: `std::unique_ptr<LogEntry>` 自动内存管理
- **移动语义**: `std::make_move_iterator`, `std::move` 提升性能
- **RAII原则**: 自动资源管理（文件流、内存等）
- **Lambda表达式**: `std::copy_if` 中的条件函数

### 2. STL 容器和算法
- **std::vector**: 动态数组存储日志条目
- **std::map**: 统计日志级别分布
- **std::regex**: 复杂日志格式解析
- **算法库**: `std::sort`, `std::copy_if`, `std::find`

### 3. 面向对象设计
- **类的封装**: 私有成员、公有接口
- **构造函数重载**: 默认构造、参数构造
- **运算符重载**: `operator<<`, `operator<`, `operator==`
- **析构函数**: 资源清理（虽然这里主要由编译器生成）

### 4. 异常处理
- **异常安全**: `try-catch` 块保护关键操作
- **自定义异常**: `std::runtime_error` 传递错误信息
- **异常传播**: 函数间的异常处理机制

### 5. 文件 I/O 和流处理
- **文件流**: `std::ifstream` 读取日志文件
- **字符串流**: `std::ostringstream`, `std::istringstream`
- **格式化输出**: `std::setw`, `std::setprecision`

### 6. 时间处理
- **chrono库**: `std::chrono::system_clock::time_point`
- **时间解析**: `std::get_time`, `std::put_time`
- **时间格式化**: 多种时间戳格式支持

### 7. 正则表达式
- **模式匹配**: 多种日志格式的识别
- **捕获组**: 提取时间戳、级别、来源、消息
- **错误处理**: `std::regex_error` 异常捕获

### 8. 命令行程序设计
- **参数解析**: `argc`, `argv` 处理
- **用户界面**: 帮助信息、错误提示
- **功能模块化**: 不同选项对应不同功能
