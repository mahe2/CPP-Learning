# VS Code C++ 快速运行指南

## 🎯 三种运行方式

### 方式1：Code Runner（推荐，最简单）✨

**已配置完成！** 现在您有三种方式运行：

1. **点击右上角的 ▶️ 运行按钮**
2. **快捷键**：`Ctrl+Alt+N`（Windows/Linux）或 `Control+Option+N`（macOS）
3. **右键菜单**：右键点击代码 → `Run Code`

**停止运行**：`Ctrl+Alt+M` 或点击右上角的 ⏹️ 停止按钮

---

### 方式2：终端命令（传统方式）

```bash
# 编译
g++ -std=c++17 01_hello_world.cpp -o hello

# 运行
./hello
```

---

### 方式3：配置任务（高级）

按 `Cmd+Shift+P`（macOS）或 `Ctrl+Shift+P`（Windows），输入 `Tasks: Configure Task`

---

## ⚙️ 已配置的设置

在 `.vscode/settings.json` 中已配置：

- ✅ 使用 C++17 标准
- ✅ 自动保存后运行
- ✅ 在终端中运行（可看到输入输出）
- ✅ 清除之前的输出
- ✅ 显示执行信息

---

## 🎨 使用示例

1. 打开任意 `.cpp` 文件（如 `01_hello_world.cpp`）
2. 点击右上角的 **▶️ 运行按钮**
3. 程序会自动编译并在终端中运行

---

## 💡 提示

- **有输入的程序**：必须在终端中运行（已配置）
- **快速测试**：直接点运行按钮
- **调试程序**：使用 VS Code 的调试功能（F5）

---

## 🐛 常见问题

### Q: 点击运行后没反应？
A: 确保文件已保存（Cmd+S）

### Q: 找不到运行按钮？
A: 检查右上角工具栏，或使用快捷键 `Ctrl+Alt+N`

### Q: 编译错误？
A: 检查代码语法，查看终端中的错误信息
